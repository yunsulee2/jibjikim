# 집지킴 🏠 — 사회초년생을 위한 AI 부동산 계약 비서

전세사기 예방과 안전한 계약을 돕는 개인 맞춤형 AI 코파일럿 웹앱.

## 주요 기능
- **맞춤 대시보드** — 잔금일 D-day, 계약 전/중/후 자동 생성 체크리스트
- **AI 등기부등본 진단** — (보증금+선순위채권)÷주택가액 기반 위험도 점수, 특약 문구 자동 생성
- **지원 제도 자동 판정** — 나이·지역·소득·계약유형 기반 자격 매칭 (버팀목, 중기청, 청년월세지원 등)
- **AI 계약 비서 챗** — 내 계약 정보를 아는 컨텍스트 기반 Q&A
- **안전 계약 가이드** — 전세사기 5대 유형, 용어 사전, 절차 타임라인

## 실행
`index.html`을 브라우저에서 열면 됩니다. 서버·빌드 불필요 (데이터는 localStorage 저장).
